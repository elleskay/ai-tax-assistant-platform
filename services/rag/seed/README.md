# Seed corpora (per workspace)

Self-authored guidance documents for the example workspaces, used to demonstrate
RAG. These contain **facts** (rates, thresholds, deadlines, which are not
copyrightable) plus **original plain-language summaries** and **citation links**
to the official IRAS pages. No IRAS content is reproduced (their Terms of Use
forbid it). Each note carries a "last verified" date.

Upload a workspace's folder via the **Documents** page (with that workspace
active), or POST them to the RAG service `/index` endpoint.

- `individual-income/` — the flagship workspace corpus.
- `corporate/` — the second workspace corpus.

> Unofficial concept demo, not affiliated with IRAS. Verify all figures against
> the cited official pages before any real use.
